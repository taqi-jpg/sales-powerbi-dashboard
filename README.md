# 📊 Sales Analysis — Power BI Dashboard

## 📌 Project Overview
Interactive 2-page Sales Dashboard built in Power BI 
with DAX measures, slicers and KPI cards.

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query

## 📊 Dashboard Pages

### Page 1 — Sales Overview
- 3 KPI Cards (Revenue, Profit, Quantity)
- Category and Month slicers
- Sales by Category bar chart
- Sales Distribution donut chart
- Top 5 Products horizontal bar
- Monthly Sales Trend line chart
- Profit by Category bar chart
- DAX measure cards (Profit Margin, Electronics Sales)

### Page 2 — Detailed Analysis
- Product Details table with conditional formatting
- Sales by Category and Month matrix
- Sales Treemap
- Filter by Category slicer

## 📈 DAX Measures Created
- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Avg Order Value
- Electronics Sales
- High Value Orders
- Jan Sales
